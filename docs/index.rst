Welcome to googlesat's documentation!
=====================================

What is googlesat?

This software is designed to download Sentinel 2 (L1C & L2A) from Google Cloud Platform and **does not require an google account and a creation of a GCP project** for data searching or downloading.
You can read more about the public Google data access `here <https://cloud.google.com/storage/docs/public-datasets/>`_ and for Sentinel-2 data `here <https://cloud.google.com/storage/docs/public-datasets/sentinel-2>`_.
